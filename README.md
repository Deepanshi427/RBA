# Role-Based Access Control (RBAC) API

## Features
- JWT Authentication
- Role-Based Access Control
- Override Access Middleware (MASTER-123)
- Admin, Manager, User routes

## Middleware
- `auth.js` → Authentication
- `rbac.js` → Normal Role-Based Access Control
- `overrideAccess.js` → Override access with x-override-key
